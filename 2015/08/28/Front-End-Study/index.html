前言

简单记录个人学习 Web 前端过程中使用的教程和书籍。仅列出较系统和专业化的资料，不包括网上零散的文章和教程。

  • 资源样式：[完成日期]. [教程标题（带链接）] [投入时间]；
  • 资源简介 + 个人学习方式 + 有关配图。

PS: 学习前端前的一些个人相关信息和经验：

  1. 非计算机类专业，文科生，编程经验基本为零；
  2. 能流畅阅读大部分英文资料；
  3. 经常折腾各种系统、软件和手机 APP；
  4. 善用利用搜索解决各类问题；
  5. 追求“秩序”，喜欢把各种事物变得井井有条。
